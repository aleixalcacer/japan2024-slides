# Fair Archetypal Analysis

This repository contains materials and code for a talk on **Fair Archetypal Analysis**. The goal of this project is to explore and present methods for performing archetypal analysis with a focus on fairness, ensuring that the resulting models and insights are equitable and unbiased.

## Contents
- Slides and presentation materials
- Example code and notebooks
- References and further reading

## About Fair Archetypal Analysis
Archetypal Analysis is a technique for representing data as mixtures of extremal points (archetypes). This talk discusses how to adapt archetypal analysis to address fairness concerns, such as mitigating bias and ensuring fair representation across groups.
